Editor Overview

Integrate the editor in your application and offer digital resumes from within your website or blog. You can use any number of Getboarded templates. The iframe embed code gives full functional editor to your users with this integration. You may use the link to launch the editor from any page.

Getting Started Resources

  • You can choose size of the editor to implement, the editor works in website mode, tablet mode and mobile mode seamlessly.
  • Admin email chosen will provide you with access to all resumes created through the implemented editor when you login to Getboarded.com
  • You can order a custom design and have your own resume created from within your platform
